# Privacy Policy

This privacy policy describes how personal health and activity data is collected, used, and stored by this application.

## Data Collection
The application collects health and activity data provided by Garmin through the Garmin Health API, including but not limited to sleep, stress, heart rate, HRV, body battery, and activity summaries.

## Purpose of Data Use
The collected data is used solely to provide personal wellness insights, trend analysis, and data visualizations to the user.  
The application is intended for informational and wellness purposes only and does not provide medical advice, diagnosis, or treatment.

## Data Sharing
Personal data is not sold, shared, or disclosed to third parties. Data is only accessible to the authenticated user.

## Data Storage and Security
Data is stored securely using industry-standard security practices. Access to data is restricted and protected through authentication and encrypted connections where applicable.

## Data Retention
Users may request deletion of their data at any time. Data is retained only as long as necessary to provide the service.
